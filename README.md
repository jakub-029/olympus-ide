# Olympus IDE

A desktop IDE with an integrated terminal, code editor, file explorer, and embedded browser preview. Built with Electron, React, and TypeScript.

## Develop

```bash
npm install
npm run dev
```

## Shortcuts

- `⌘K` / `⌘P` — Command palette
- `⌃\`` — Toggle terminal
- From the palette: toggle theme (Thunder / Marble), toggle preview, new file
